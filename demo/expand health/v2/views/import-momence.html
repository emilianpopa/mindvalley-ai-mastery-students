Enrich with Sales/Bookings Data

Upload the "Total Bookings" report from Momence to add payment details, prices, and membership info to existing appointments.

How to get this report:

  1. In Momence, go to Reports → Total Bookings (or Sales by Appointment)
  2. Set your date range and export as CSV
  3. Upload the CSV below to match sales data with appointments

This will update: Sale Value (price), Payment Method, and Membership used

Fix Appointment Durations

If appointments are showing incorrect durations (e.g., 90-minute sessions appearing as 60 minutes), click this button to recalculate all appointment durations based on their service names.

How it works: The system will parse each appointment's title/service name to extract the correct duration (e.g., "HBOT1 90 min" → 90 minutes) and update the end times accordingly.

Fix Staff/Practitioner Assignments

If appointments are showing "Not assigned" for practitioner, use this to match practitioners from your Momence CSV.

How it works:

  1. Upload the same Momence "Appointment Reservations" CSV used for import
  2. The system will match practitioners by name to your existing staff records
  3. Appointments will be updated with the correct staff ID
